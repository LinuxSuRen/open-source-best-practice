    还有一些其他的比较推荐的习惯、方法：

    • 习惯查看邮件
      • 一定要把自己常用的邮箱地址关联到 GitHub 上，这样才能及时收到 issues 和 PR 中的互动信息
    • 通过电脑、手机来管理自己的日程
      • 开源社区总会有各种各样的会议，如果没有使用日历的习惯的话，非常容易遗忘重要的事情
